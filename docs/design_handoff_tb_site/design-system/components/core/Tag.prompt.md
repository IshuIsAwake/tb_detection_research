Monospace chip for configs, hyperparameters, capabilities and identifiers. Quieter than `Badge` — use it for technical literals, not status.

```jsx
<Tag>512 @ batch 16</Tag>
<Tag tone="primary">mosaic</Tag>
<Tag tone="muted">CC BY 4.0</Tag>
```

- **tone**: `neutral` (default) · `primary` · `muted`.
- Set in mono with tabular figures; keep contents short and literal.
