Foundational papers for architecture decisions in this project. A reference list, not a review — titles, links, and a one-line note on why each matters. Click ▼ on any row for a short description.

Reading order — papers build on each other; read in dependency order, not by date:
  1. Backbones first — ResNet → FPN. They sit under everything downstream.
  2. Two detector chains in parallel — two-stage (R-CNN → Faster R-CNN) and one-stage (YOLOv1 → RetinaNet). This is the YOLO vs RetinaNet vs Faster R-CNN decision.
  3. Domain ground-truth — TBX11K + CheXNet + MoCo-CXR.
  4. Pretraining — BYOL / DINO, when returning to the Kaggle SSL decision.
  5. Next phase — Attention → ViT → Swin, then S4 → Mamba → Vision Mamba.
